# SKILL.md - telegram-mini-app

## Purpose
Telegram Mini App Integration - Interaktive UI-Elemente für Telegram-Interaktionen.

## What It Does
- Bietet Button-Interfaces für Telegram
- Ermöglicht interaktive Umfragen
- Unterstützt Rich-Media-Präsentation
- Integriert mit Telegram Bot API

## Usage
```bash
# Mini-App starten
./telegram-mini-app.sh start

# Button-Menü senden
./telegram-mini-app.sh buttons "<nachricht>" <button1> <button2> ...

# Umfrage erstellen
./telegram-mini-app.sh poll "<frage>" <option1> <option2> ...
```
