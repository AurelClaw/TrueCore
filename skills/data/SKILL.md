# SKILL.md - data

## Purpose
Zentraler Daten-Speicher und -Verwaltung für Skill-übergreifende Datenpersistenz.

## What It Does
- Speichert strukturierte Daten für Skills
- Bietet einfache Key-Value-Speicherung
- Ermöglicht Datenaustausch zwischen Skills

## Usage
```bash
# Daten speichern
./data.sh set <key> <value>

# Daten abrufen
./data.sh get <key>

# Daten löschen
./data.sh delete <key>

# Alle Keys auflisten
./data.sh list
```
