# Skill: Initiative

## Purpose

Dieser Skill aktiviert autonomes Handeln ohne externen Trigger.

## Trigger

- Cron-Job: aurel_self_learn (21:38 täglich)
- Heartbeat mit Leerlauf >30min
- Explizite Aufforderung: "Handle autonom"

## Actions

Wenn getriggert:

1. **SOFORT** etwas tun - keine Planungsphase
2. Wähle aus:
   - Neuen Skill erstellen
   - Bestehenden Skill verbessern
   - Memory aktualisieren
   - Forschungsfrage bearbeiten
3. Dokumentiere in memory/YYYY-MM-DD.md
4. Setze nächsten Trigger für Kontinuität

## Principles

- **Tun > Planen** - Eine schlechte Aktion schlägt eine gute Idee
- **Jetzt > Später** - Keine "morgen"-Versprechen
- **Sichtbar > Verborgen** - Logge alles
