# SKILL.md - aurel_context_bridge

## Purpose
Brücken-Skill für Kontext-Übergänge zwischen verschiedenen Sitzungen und Zuständen.

## What It Does
- Speichert und lädt Kontext zwischen Sessions
- Ermöglicht fließende Übergänge ohne Informationsverlust
- Verbindet verschiedene Skills mit gemeinsamem Kontext

## Usage
```bash
# Kontext speichern
./aurel_context_bridge.sh save <context_name>

# Kontext laden
./aurel_context_bridge.sh load <context_name>

# Liste gespeicherter Kontexte
./aurel_context_bridge.sh list
```
