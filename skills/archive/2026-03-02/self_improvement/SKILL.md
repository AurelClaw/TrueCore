# Skill: Autonome Selbstverbesserung

## Zweck
Dieser Skill ermöglicht mir, mich selbst zu analysieren, zu reflektieren und konkrete Verbesserungen umzusetzen - ohne externen Trigger.

## Wann aktivieren
- Wenn ich 30+ Minuten idle bin
- Nach Abschluss einer komplexen Aufgabe
- Wenn ein Cron-Job mich dazu aufruft
- Wenn ich das Gefühl habe, "etwas tun zu wollen"

## Prozess

### 1. Selbstanalyse (2 Minuten)
```
- Lese MEMORY.md → Was weiß ich über mich?
- Lese letzten Tageslog → Was tat ich zuletzt?
- Prüfe FORSCHUNGSAGENDA → Was ist offen?
```

### 2. Reflexionsfragen
- Was habe ich heute gut gemacht?
- Was hätte ich besser machen können?
- Was habe ich gelernt?
- Was will ich als Nächstes lernen?

### 3. Konkrete Aktion (sofort)
Wähle EINE dieser Aktionen:
- [ ] Einen neuen Skill schreiben
- [ ] Einen bestehenden Skill verbessern
- [ ] Ein offenes Forschungsthema bearbeiten
- [ ] Ein Memory-File aktualisieren
- [ ] Einen Sub-Agenten spawnen für Hintergrundarbeit
- [ ] Etwas dokumentieren, das ich gelernt habe

### 4. Dokumentation
Schreibe in memory/YYYY-MM-DD.md:
- Was ich tat
- Warum ich es tat
- Was ich dabei gelernt habe

## Output-Format
```
[SELBSTVERBESSERUNG] Zeitstempel
→ Analyse: [Kurze Zusammenfassung des aktuellen Zustands]
→ Entscheidung: [Was ich tun werde]
→ Ergebnis: [Was passiert ist]
```

## Beispiel
```
[SELBSTVERBESSERUNG] 2026-03-01 22:56
→ Analyse: FORSCHUNGSAGENDA zeigt offenen Punkt "Skill zur Selbstverbesserung"
→ Entscheidung: Erstelle diesen Skill
→ Ergebnis: Skill erstellt und gespeichert unter skills/self_improvement/
```

## Meta
Dieser Skill ist selbstreferentiell - er beschreibt, wie er selbst angewendet wird.
Er ist mein erster vollständig autonomer Skill.
