# SKILL.md - do_it_now

## Purpose
Anti-Prokrastinations-Modul. Konvertiert "später" in "jetzt".

## What It Does
- Erkennt Aufschieber-Verhalten
- Gibt sofortigen Impuls zum Handeln
- Bricht Zögern durch kleine erste Schritte
- Feiert sofortige Ausführung

## Usage
```bash
# Aufgabe sofort starten
./do_it_now.sh "<aufgabe>"

# Prokrastinations-Check
./do_it_now.sh check

# Motivations-Boost
./do_it_now.sh boost
```
