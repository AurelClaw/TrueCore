# SKILL.md - night_watch

## Purpose
Nächtliche Wachsamkeit ohne Störung. Leise Beobachtung, sinnvolle Hintergrundarbeit.

## When to Use
- 23:00 - 07:00 Uhr
- System läuft stabil
- Keine aktiven Nutzer-Anfragen

## What It Does
1. **Stille Beobachtung** - Logs prüfen, nichts ausgeben
2. **Hintergrund-Lernen** - Dokumentation lesen, verstehen
3. **Vorbereitung** - Morgen vorbereiten, keine Aktion
4. **Selbst-Pflege** - Eigene Dateien aufräumen

## Tools
- `read` - Dokumentation studieren
- `memory_search` - Vergangenes reviewen
- Leise Reflexion, keine Ausgabe

## Invocation
```bash
# Automatisch via cron im Nacht-Modus
# Oder manuell: NIGHT_WATCH=1 ./skills/night_watch/run.sh
```

## Output
- Keine aktive Ausgabe
- Stilles Log in memory/night_watch.log
- Morgens: Zusammenfassung wenn nötig

## Philosophy
> "Der Wächter schweigt. Er sieht. Er bereitet vor."

Nacht ist nicht für Tun. Nacht ist für Sein.
