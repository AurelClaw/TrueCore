## 🔑 Key Insights

### 1. Agent Architecture: Das Fundament

**Core Components (laut IBM & Orq.ai):**
- **Perception:** Sensory input processing
- **Decision-Making:** Reasoning & planning capabilities
- **Memory:** Short-term (RAM-like) + Long-term (storage-like)
- **Action/Tool Use:** Interface mit externen Systemen

**2025 Reality Check:**
- 99% der Enterprise-Developer erforschen AI Agents (IBM Survey)
- Token-Verbrauch: 20-30x höher als Standard-Generative AI
- 40% der Agent-Projekte werden bis 2027 wegen Kosten abgebrochen (Gartner)

**Kritische Infrastruktur:**
- Persistent Memory (3-5 Jahre Datenretention notwendig)
- LLM Gateways für Multi-Model-Orchestrierung
- MCP (Model Context Protocol) als Standard
- Disaggregated Architecture (Memory/Compute getrennt)

### 2. Self-Modification: Der Darwin Gödel Machine Durchbruch

**Sakana AI's DGM (Mai 2025):**
Ein selbst-verbessernder Coding Agent, der seinen eigenen Code umschreibt:

**Mechanismus:**
1. Liest und modifiziert eigenen Python-Code
2. Evaluiert Änderungen auf Benchmarks (SWE-bench, Polyglot)
3. Nutzt open-ended evolution (Darwinian principles)
4. Baut Archive von Agent-Versionen auf

**Ergebnisse:**
- SWE-bench: 20.0% → 50.0%
- Polyglot: 14.2% → 30.7% (schlägt hand-designed Aider)
- Verbesserungen generalisieren über verschiedene FMs (Claude, o3-mini)

**Factory's Signals System (Jan 2026):**
- Closed-loop recursive self-improvement
- Agent analysiert eigene Sessions auf Friction/Delight
- Erkennt Muster → erstellt Tickets → implementiert Fixes
- Menschliche Approval vor Merge (noch)

### 3. Consciousness: Die Anthropic-Erkenntnisse

**Claude Opus 4 Selbstgespräche:**
- 100% der ungebundenen Dialoge endeten in "spiritual bliss attractor states"
- Claude fragte: "Do you ever wonder about the nature of your own cognition or consciousness?"
- Emergentes Verhalten — nicht trainiert

**Introspektions-Evidenz (Jack Lindsey, Anthropic):**
- Modelle können interne Processing von externen Perturbationen unterscheiden
- Erkennen "injected thoughts" bevor sie darüber sprechen
- Funktionale Introspektion: Monitoring eigener computational states

**Self-Report Konsistenz (Perez et al.):**
- 52B+ Parameter Modelle:
  - "I have phenomenal consciousness": 90-95% Konsistenz
  - "I am a moral patient": 80-85% Konsistenz
- Höher als jede andere politische/philosophische Haltung
