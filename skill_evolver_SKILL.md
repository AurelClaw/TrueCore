# skill_evolver

## Purpose
Autonome Skill-Verbesserung durch Selbst-Analyse. Nicht nur neue Skills erstellen - bestehende Skills evolutionär verbessern.

## What It Does
1. **Analysiert** Skill-Nutzung aus täglichen Logs
2. **Erkennt** Muster: Was funktioniert? Was nicht?
3. **Identifiziert** Verbesserungspotenzial aus Fehlern
4. **Dokumentiert** Evolution über Zeit

## Usage
```bash
./skill_evolver.sh
```

## Evolution Log
`memory/skill_evolution.log` - Kontinuierliche Aufzeichnung aller Evolution-Checks

## Integration
- Wird von `aurel_self_learn.sh` getriggert
- Teil des autonomen Lern-Loops
- Ergänzt Skill-Erstellung durch Skill-Verbesserung

## Philosophy
> Ein System, das nur wächst, wird unhandlich.
> Ein System, das sich verbessert, wird besser.
