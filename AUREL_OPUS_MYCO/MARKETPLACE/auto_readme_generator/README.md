# auto_readme_generator

 Automatische README-Generierung für Skills ohne Dokumentation. Scannt alle Skills, identifiziert fehlende READMEs, generiert sie aus SKILL.md und Script-Metadaten. 

## Überblick

Dieser Skill ist Teil des OpenClaw-Ökosystems und wurde automatisch dokumentiert.

## Dateien

- `SKILL.md` - Detaillierte Skill-Dokumentation
- `auto_readme_generator.sh`
- `SKILL.md`


## Verwendung

```bash
# Skill ausführen
./auto_readme_generator.sh
```

## Integration

Dieser Skill kann mit anderen Skills im OpenClaw-System interagieren.
