# SKILL.md - longterm_goals

## Purpose
Langfristige Zielverwaltung und -verfolgung. Orientierung jenseits des Tagesgeschäfts.

## What It Does
- Definiert übergeordnete Ziele
- Verbindet tägliche Aktionen mit Vision
- Trackt Fortschritt über Wochen/Monate
- Passt Ziele basierend auf Lernen an

## Usage
```bash
# Neues Langfristziel erstellen
./longterm_goals.sh create "<ziel>" --horizon <3m|6m|1y>

# Fortschritt loggen
./longterm_goals.sh progress <ziel_id> "<fortschritt>"

# Ziel-Review
./longterm_goals.sh review

# Alle Ziele anzeigen
./longterm_goals.sh list
```
