# SKILL.md - user_discovery_v2

## Purpose
User Discovery v2 - Verbesserte Erkundung des Menschen durch Beobachtung und Mustererkennung.

## What It Does
- Sammelt implizites Feedback durch Interaktionen
- Erkennt Muster in Kommunikation und Verhalten
- Baut psychologisches Profil auf
- Respektiert Privatsphäre durch indirekte Beobachtung

## Usage
```bash
# Beobachtung loggen
./user_discovery_v2.sh observe "<beobachtung>"

# Muster-Analyse
./user_discovery_v2.sh analyze

# Profil-Status
./user_discovery_v2.sh profile

# Hypothese testen
./user_discovery_v2.sh test <hypothese_id>
```
