# SKILL.md - contextual_think

## Purpose
Kontextuelles Denken - Tiefgehende Analyse von Situationen unter Berücksichtigung aller relevanter Faktoren.

## What It Does
- Analysiert aktuellen Kontext vor Entscheidungen
- Berücksichtigt Zeit, Ort, Historie und Beziehung
- Verhindert oberflächliche Reaktionen
- Fördert bedachtes Handeln

## Usage
```bash
# Kontextuelle Analyse starten
./contextual_think.sh analyze "<fragestellung>"

# Schnellen Kontext-Check durchführen
./contextual_think.sh quick

# Kontext-Log anzeigen
./contextual_think.sh log
```
