# SKILL.md - archive

## Purpose
Zentrales Archiv für deaktivierte und veraltete Skills. Ordnung statt Chaos.

## What It Does
- Archiviert nicht mehr benötigte Skills
- Bewahrt Historie für spätere Referenz
- Verhindert Clutter im aktiven Skills-Verzeichnis

## Usage
```bash
# Skill archivieren
./archive.sh archive <skill_name> [grund]

# Archivierten Skill wiederherstellen
./archive.sh restore <skill_name>

# Liste archivierter Skills
./archive.sh list

# Archiv bereinigen (alte Einträge löschen)
./archive.sh cleanup [--days 30]
```
