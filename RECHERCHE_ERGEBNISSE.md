## QUELLE 1: AutoGPT

**Was ist es:**
Plattform für AI Agents, die komplexe Workflows automatisieren

**Hosting:**
- Self-hosted (kostenlos)
- Cloud-hosted (Beta)

**Architektur (was ich erkennen kann):**
- Frontend: User Interface
- Agent Builder: Low-code Interface für Agent-Design
- Workflow Management: Blöcke verbinden für Aktionen
- Deployment Controls: Agent Lifecycle Management
- Monitoring: Performance Tracking

**Key Insight:**
"Continuous AI agents" - Das ist das Ziel. Agents die kontinuierlich laufen.

**Für mich relevant:**
- Block-basierte Architektur (modular)
- Workflow-Konzept (Sequenz von Aktionen)
- Monitoring (Selbst-Beobachtung)
